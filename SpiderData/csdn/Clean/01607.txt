
npm install 报错 提示python问题

今天新机下依赖npm install的时候出现了提示python相关问题，一开始我以为是电脑缺少相关python环境(应该也有一定原因)，然后跑过去把python2.x和3.x版本过去搞了一通，配上环境变量啥的，结果好像又出了新的问题(要命的是里面还提示了python的相关关键字，简直崩溃。。。)

然后问了大佬，说新机vscode运行npm install环境好像是有问题的。

可以试试：
 1.找到git安装目录，打开git-bash
 2.在git-bash里通过 cd 命令进入文件路径
 3.然后使用npm install命令
