
python引入opencv出错解决方法


问题

import cv2时出错：
 Install packages failed: Installing packages: error occurred.

opencv下载地址：
 https://www.lfd.uci.edu/~gohlke/pythonlibs/


python pip命令安装

选择（即便是64位也选这个，不要选amd64）：
 opencv_python‑3.4.6‑cp37‑cp37m‑win32.whl

安装命令
 pip install opencv_python-3.4.6-cp37-cp37m-win32.whl


anaconda pip安装

运行Anaconda Prompt

执行安装命令
 pip install opencv_python-3.4.6-cp37-cp37m-win_amd64.whl
