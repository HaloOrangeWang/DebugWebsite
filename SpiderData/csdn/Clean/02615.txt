从pip 里安装发现如下出错：

要解决这个问题，只能到网站：

下载地址：http://www.lfd.uci.edu/~gohlke/pythonlibs/

下载到文件：Twisted-17.5.0-cp36-cp36m-win_amd64.whl

然后安装：pip install Twisted-17.5.0-cp36-cp36m-win_amd64.whl

这样才可以安装成功。

最后就可以安装： pip install scrapy

在运行的过程中，需要pywin32, 可以使用下面命令安装：

pip install pypiwin32
