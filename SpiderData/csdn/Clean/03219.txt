
1、安装必要Python库

打开命令行窗口（同时按 Win+R，输入cmd ）

输入：pip install flake8 回车（用于代码错误提醒）

输入：pip install autopep8 回车（用于代码格式化）

pip install flake8
pip install autopep8

pip install flake8
pip install autopep8


2、调整VsCode设置

在 VScode 中打开设置，搜索框输入 python.linting.flake8enabled  回车

在 Settings 界面中勾选 Whether to lint Python files using flake8  如下图

辅助说明：【VsCode】格式化快捷键 Alt+Shift+F

最后的善意提醒：如果没有切实需求，可以不安装 flake8 库，因为安装之后会给你报很多错误，包括代码不规范等~
