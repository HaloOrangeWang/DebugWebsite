
pip install xxxx失败的情况

很多情况下pip install xxxx失败的情况下，可以到https://www.lfd.uci.edu/~gohlke/pythonlibs/下载相应的.whl文件，然后打开cmd，输入pip install xxx.whl即可。

例如：pip install python-Levenshtein出错，可以到上述网站下载python_Levenshtein‑0.12.2‑cp37‑cp37m‑win_amd64.whl,其中cp指的是python的版本。
 
 然后打开cmd，输入pip install python_Levenshtein‑0.12.2‑cp37‑cp37m‑win_amd64.whl即可。
