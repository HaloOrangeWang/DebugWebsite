3-定位print问题-就是这个问题！

原因剖析：有时候进程的运行环境里，locale 会被设置成只支持 ASCII 字符集的（比如 LANG=C）。这时候 Python 就会把标准输出和标准错误的编码给设置成 ascii，造成输出中文时报错。在这里的 Visual Studio Code 编辑器中就被设置成了ascii编码，造成输出中文报错。
