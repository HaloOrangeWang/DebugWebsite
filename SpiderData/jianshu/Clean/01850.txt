sudo pip3 install packagename

代表进行全局安装，安装后全局可用。如果是信任的安装包可用使用该命令进行安装。

pip3 install --user packagename

代表仅该用户的安装，安装后仅该用户可用。处于安全考虑，尽量使用该命令进行安装。
