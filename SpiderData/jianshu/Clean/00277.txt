public static String setAgreementRedis(String key, String value, int seconds) {   JedisPool pool = StockRedisUtil.getPool();   Jedis jedis = null;   try {      if (pool != null) {         jedis = pool.getResource();         String eval2 = jedis.setex(key, seconds, value);         return eval2;      }   } catch (Exception e) {      jedis.close();      e.printStackTrace();   } finally {      StockRedisUtil.returnResource(pool, jedis);   }   return null;}

项目jedis.close()时报错，网上搜到的方法不能解决，检查后发现，redis配置未设密码引起此报错。

spring.redis.host=127.0.0.1

spring.redis.port=6379

spring.redis.password=123456

加上密码后，成功启动项目。
