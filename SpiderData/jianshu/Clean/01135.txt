在安装appium环境时，需要安装 Python 依赖 pip install Appium-Python-Client，我安装时Python版本是3.5。

1、出现这个问题的原因是pip21.0.1版本在2020年1月停止支持Python2，在2020年3月停止支持Python3。

2、点击进这个链接下载get-pip.py文件，下载之后我是放在了图片上这个位置。Index of /pip/ (pypa.io)

3、接着执行命令py get-pip.py,安装成功之后就可以执行下一步了。

我这个是在windows环境下的命令，如果是Linux或mac的环境下，命令是不一样的。

python get-pip.py
