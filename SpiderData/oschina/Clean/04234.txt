
python 安装pytorch 及 安装失败解决办法

　　[转] pytorch安装失败解决办法

　　[转] 一分钟在win10终端成功安装pytorch

　　pytorch 的安装方法有2种，一种是pip安装，另一种就是下载whl文件然后本地安装。


　　方法一：pip安装

　　或许会想到直接使用

pip install torch

　　但是这样会报错，安装失败。所以，可以到【官网】找对应版本进行下载。

　　　　1. 点击进入官网

　　　　2. 点击Get Started，按照自己的配置选择相应的选项，最后复制生成的命令到终端，鼠标右键粘贴，回车执行就实现安装了。

　　　　由于我没有安装任何库，所以它会自动把所需要的依赖库都下载下来。

　　　　3. 测试有没有成功

　　　　居然显示import error！！！

　　　　幸好，在网上找到解决方法，发现是numpy版本的问题，换一个numpy库就行了。

　　　　在【此网站】中下载对应的numpy+mkl包到对应目录下，然后在终端中执行下面命令即可进行安装。我下载的是 "numpy-1.16.5‑cp37‑cp37m‑win_amd64.whl" 。

pip install 文件名

　　　　4. 再重新测试一下：

Perfect !


　　方法二：本地安装

　　　　1. 在【此网站】下载对应的numpy+mkl包，方法同上，使用 pip 命令安装

　　　　2. 在【此网站】下载对应的torch包

　　　　3. 测试一下

Perfect !
