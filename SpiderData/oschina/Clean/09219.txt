注：此文写成于2019-06-08


错误信息

Exception in thread "main" java.lang.Error: Unresolved compilation problem: Syntax error on token "}", delete this token


解决方法

我在 main 方法前发现一个没有写完整甚至写错了的方法。在我将那两行代码删除/注释以后，报错就没有了。完美解决。

其他参考解决办法：

将代码复制到记事本，再新建一个类，将代码粘贴回来。

新建一个项目，将报错的类复制到新类里面。（有时可以解决项目报感叹号的问题） 以上玄学的解决方法，我还不清楚是什么原因。


结论

Eclipse自动生成代码时出错了！ 代码没有生成完整 ，很多时候是多了或者少了一个结尾的花括号“}”。 仔细观察代码，查看每一个方法是否完整，手动修改正确或者删除即可解决报错。
