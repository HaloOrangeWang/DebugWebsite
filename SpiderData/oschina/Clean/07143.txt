Linux系统常见英文报错

英文报错

中文释义

command not found

该命令没有找到

No such file or directory

没有这个文件或目录

Permission denied

权限不足

No space left on device

磁盘没有剩余空间

File exists

该文件已经存在

Is a directory

这是1个目录

Not a directory

不是1个目录

Warning: Changing a readonly file

警告：正在改变一个只读文件

Found a swap file by the name  ".5.swp"

发现了一个swap文件名字为5.swp(vim编辑器碰到的意外关闭文件有缓存文件)

unrecognized option '--sfasf'

不识别的选项，请查看帮助

No route to host

没有路由到主机（防火墙没关）

connection refused

连接拒绝（防火墙没关或服务端口没开）

Access denied

访问拒绝（权限或者selinux）
