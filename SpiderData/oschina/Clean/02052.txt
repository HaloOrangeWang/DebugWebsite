一、安装wheel库

命令：pip install wheel


二、下载whl文件

whl文件相当于一个压缩包，whl文件可以在下方的网址中下载

Python Extension Packages for Windows

在选择whl的时候先Ctrl + F搜索一下gmpy2

python版本为3.8就选择cp38
 64位系统选择win_64，32位系统选择win32

我的环境是python3.8、windows 64位，对应的文件就是gmpy2‑2.0.8‑cp38‑cp38‑win_amd64.whl


三、安装whl

找到你下载的whl文件路径并复制

命令：pip install [whl路径]


四、安装gmpy2

命令：pip install gmpy2
