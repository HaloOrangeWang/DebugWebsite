1、报错内容：ModuleNotFoundError: No module named 'pytest'，

分析问题，

<1、进入settings，搜索pytest，安装即可。

<2、若安装后继续报错可能是当前文档与安装的python版本不对应，适当降低版本。

2、ImportError: No module named 'yaml',需要安装yaml配置文件，python3的安装方法，进入python的路径

安装：pip3 install pyyaml，必须通过cmd命令安装，

3、普通安装类报错：No module named 'pymysql'，进入设置，选择安装的配置，点Install Package,即可安装成功
