问题与需求：用python下载第三方库或包的时候出错怎么办？

方法一般有以下四种，可以解决大部分的问题。

1、在cmd命令控制窗口输入要安装的包即可：

pip install ...
... 为所需的第三方包，如pygame等

可能有些时候因为网络或者其他的问题，会显示下载失败，此时用2、3方法。

2、去python的pypi官网下载对应名称包或者库的.whl文件保存在本地。

3、若官网访问太慢或者下载失败，可采用国内镜像下载相关第三方包：

目前有很多企业和组织都做了镜像服务器，上面存放了绝大多数所需要用到的第三方包或者库，比如清华镜像、腾讯镜像等。

这里我比较喜欢使用阿里镜像，速度快且稳定。

在浏览器输入：

https://mirrors.aliyun.com/pypi/simple/
注意simple/后面加上自己所需的第三方包或库的名称，如：numpy等

点击下载对应自己python和操作系统（winx64或32、linux）版本的.whl文件，保存在本地目录。

4、若下载的是tar.gz文件形式的第三方库：

只需要将文件解压后，cmd命令行cd到相应目录，输入：

python.exe setup.py install
即可

测试：最后打开python编辑器shell，输入：

import ...
...为所需的相应模块

若没有报错，则恭喜你成功安装此模块！

也可以在cmd窗口输入命令:

pip list
...
则会显示所有已安装的模块
