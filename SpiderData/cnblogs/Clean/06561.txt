
python3 包管理 pip 命令

pip：在线安装第三方包、管理第三方包

安装包：pip install 库名

卸载包：pip uninstall 库名

升级包：pip install -U 库名

查看当前pip版本：pip --version

列出已安装的包：pip list

生成 requirements.txt: pip freeze > requirements.txt

安装 requirements.txt: pip install -r requirements.txt

我们在cmd 中输入 where pip  或 pip -V，如果没有正常返回则是环境变量配置有误
